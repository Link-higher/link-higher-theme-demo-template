== Description ==
Link Higher is a modern WordPress blog theme focused on readability and performance. It includes support for the block editor and popular page-builders, responsive layouts, custom logo, and basic theme options via the Customizer.

== Installation ==
1. Upload the `link-higher` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance > Themes' menu in WordPress.
3. To use full-width builder layouts, edit a Page and select the "Full Width (Page Builder)" template.
4. Customize footer text, legal links, author box, and accent color in 'Appearance > Customize'.

== Frequently Asked Questions ==
= How do I enable the partner footer iframe? =
For security and WordPress.org compatibility the remote partner footer is disabled by default. If you trust the remote content, enable it in 'Appearance > Customize > Footer'.
