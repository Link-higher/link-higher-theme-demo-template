## üÜò Quick Help

### Can't find Modern Layout?
1. Go to WordPress Admin ‚Üí Appearance ‚Üí Customize
2. Look for "Theme Layouts" section
3. Check "Header Layout" dropdown
4. Modern Layout should be third option

### Assets not loading?
1. Check DevTools (F12) ‚Üí Network tab
2. Look for style.css and main.js
3. Should show status 200 (green)
4. If 404 or missing, check file paths in functions.php

### Dark mode not saving?
1. Open DevTools Console (F12)
2. Type: `localStorage.getItem('modern_layout_theme')`
3. Should show 'light' or 'dark'
4. If empty, localStorage might be disabled

### Mobile menu not working?
1. Check DevTools Console for JS errors (red text)
2. Verify main.js file loaded in Network tab
3. Test in incognito/private window
4. Try different browser
